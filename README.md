# 🇮🇳 Incredible India Project

A simple and interactive website showcasing the beauty of India including tourism places, states, temples, and culture.

---

## 🌟 Features

- 🏝️ Top tourist places in India
- 🗺️ States and Union Territories
- 🛕 Famous temples
- 📸 Beautiful images of destinations
- ⚡ Dynamic content using JSON & JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- JSON
- GitHub Pages (for hosting)
